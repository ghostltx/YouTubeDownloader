# YouTube Video Downloader

A simple and easy-to-use YouTube video downloader application.

## Features

- 🎯 Download YouTube videos with one click
- 📁 Custom save location
- 🎬 Multiple quality options (Best, 1080p, 720p, 480p)
- 📊 Download progress display
- 🔧 Built-in FFmpeg support
- 🖥️ No external dependencies required

## System Requirements

- Windows 7 or later
- No Python or other software required

## How to Use

1. **Run the application**
   - Double-click `YouTubeDownloader.exe`

2. **Download a video**
   - Copy the YouTube video URL
   - Paste it into the URL field
   - Choose save location (default: Desktop)
   - Select video quality
   - Click the green **DOWNLOAD** button

3. **Wait for download**
   - Progress bar shows download status
   - Video will be saved to your chosen location

## Troubleshooting

### SSL Error
If you encounter SSL connection errors, try:
- Check your internet connection
- Disable VPN/Proxy if using one
- Try again later

### Download Failed
- Make sure the video is not private or region-locked
- Check your internet connection
- Try a different video quality

## Technical Details

- Built with Python + Tkinter
- Uses yt-dlp for video downloading
- Includes FFmpeg for video processing
- Packaged with PyInstaller
